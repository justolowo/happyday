6.2 Question 2

This is Exercise 3 in Chapter 7 of the Textbook [R].

6.2.1 Problem

The distribution of the t-statistic for testing the significance of a correlation coefficient has n â 2 degrees of freedom. If the sample size is 36 and Î± = 0.05, what is the smallest absolute value a correlation coefficient must have to be significant? What if the sample size is 80?

6.2.2 Solution

n=36

n <- 36
r.36 <- round(2/sqrt(n),3)
# r <- 0.3333

t.critical <- qt(0.975,df = n - 2)  ## = 2.0322
t <- r.36*sqrt(n-2)/sqrt(1-r.36^2)  ## = 2.0615

When the sample size is 36, the smallest absolute value a correlation coefficient must have to be significant is 0.333

n=80

n <- 80
r.80 <- 2/sqrt(n)
# r <- 0.2236

t.critical <- qt(0.975,df = n - 2)  ## = 1.9908
t <- r.80*sqrt(n-2)/sqrt(1-r.80^2)  ## = 2.0261

When the sample size is 80, the smallest absolute value a correlation coefficient must have to be significant is 0.2236068

Calculate Pearsonâs correlation coefficient using the equation below: \[r=\frac{\sum_{n}^{i=1}(x_i-\bar{x})(y_i-\bar{y})}{(n-1)s_{x}s_{y}}\]

Solution 9c

df$speed.z <- scale(df$speed)

6.9.5 Problem 9e

Calculate Spearmanâs rank correlation coefficient using the equation below: \[r_{S} = 1 - \frac{6\sum_{i=1}^{n}d_{i}^{2}}{n^3-n}\]

Solution 9e

df$speed.r <- rank(df$speed,ties.method = "average")
df$dist.r <- rank(df$dist,ties.method = "average")

The Spearmanâs rank correlation coefficient is 0.8308283
